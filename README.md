
使用node做服务器，前端angularjs，实现高德地图上的多点标记

1,http://lbs.amap.com/ 获取key，然后在test.html中添加上

2,ifconfig 获取本机测试ip，然后在webController中加上本机的ip

3,在service中，执行start.js  >node start.js

4,在url地址栏中：ip:9909/test.html
